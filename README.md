# 🔐 Генератор паролей Pro

Генератор безопасных паролей с оценкой надежности и историей.

## Возможности
- Выбор длины пароля (6-32 символа)
- Настройка типов символов
- Оценка надежности
- История последних 10 паролей
- Копирование в буфер обмена

## Как пользоваться
1. Выберите длину пароля
2. Отметьте нужные символы
3. Нажмите "Сгенерировать пароль"
4. Скопируйте кнопкой с иконкой 📋
